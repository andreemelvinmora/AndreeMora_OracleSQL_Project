# Oracle SQL Database Project

This project contains the SQL scripts I created for my Enterprise Architecture and Database coursework. It includes database schema creation, data insertion, and analytical SQL queries executed in Oracle SQL Developer connected to an AWS-hosted Oracle instance.

## Project Contents
- **AndreeMora_Project4.sql** – Full SQL script including:
  - DROP and CREATE TABLE statements
  - INSERT statements to populate tables
  - JOIN, GROUP BY, HAVING, and subquery examples

## Skills Demonstrated
- Oracle SQL Developer
- Database schema design
- DDL and DML scripting
- Query optimization and testing
- AWS RDS Oracle connectivity
